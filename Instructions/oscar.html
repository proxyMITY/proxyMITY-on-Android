Introduction to Project Oscar

The primary objective of Project Oscar is to build the large repository of web based, interactive animated learning objects (LOs) for teaching and learning undergraduate and post graduate science and technology subjects. These self learning interactive objects will help educators to improve their teaching. These LOs will also help students in better understanding of the topic through practice and thereby improving their overall performance.

Definition of learning objects?

Leaning object is a collection of content items, practice items and assessment items created for a single learning objective. It is a new concept in IT based education that provides smaller self contained re-usable units of learning. A single learning object could be used in multiple contexts for multiple purposes.
It is not possible for any teacher to create all the learning material for the course. The reusability of learning objects offers an efficient way to facilitate instructions of commonly taught concepts, procedures, applications and skills to teachers.  These animated LOs help students in visualization of the abstract concepts.

Learning Objects on Oscar Site

The top bar of the home page of Oscar contains many tabs. Touch each tab to know more. Touch the LOs/Animations tab from the top bar and view the icons of science and engineering subjects.Touch Search tab from the top bar to search the specific LO from the repository.

Download “LO” on your tablet

Touch “Download” tab to get the list of downloadable files for each LO. Select the LO and touch the download button on the right side of the list to download all the files in the newly created folder on the tablet

Learn to use learning object

Touch a specific subject icon to find out the list of learning objects created for a particular subject.
Touch icon of the learning object of your interest, e. g Buffon’s Needle
Touch “Run” button to view all the sections of Buffon’s Needle learning object. 
Touch “Download” button to download the entire LO file.
Touch “Download IDD” button to download instructional Design document file.
Buttons on the right top corner of home page of LO represent different sections of this learning object.
Touch “Demo” button on the home page to run Buffon’s Needle demo.
Touch “Interactive” Button to learn more about the concept.
 Touch “Quiz” button to take a quiz to test your understanding of this topic.
Touch “References” button to view links for the additional relevant reference material.
Touch “Theory” button to learn more about the theoretical background of this Learning object.
